This project was made with the intent of making learning a new language a more fun and engaging process. Generally language learning services don't focus so much on the fun. This project treats it's more like a game and tracks your score so that you can try and one up your previous high scores and visualize your progress. This project uses Google Translate's API service to translate between any language! It also serves as a microservice for a teammates project by providing the translation service to them. There are also two different game modes to add more variation and rigor.
